# 💰 CryptoHub - Полнофункциональный сайт про криптовалюты

Современный, красивый и функциональный сайт для отслеживания криптовалют, чтения новостей и управления портфелем.

## 🌟 Особенности

### 📊 Главная страница
- Привлекательный дизайн с градиентами
- Информация о преимуществах криптовалют
- Быстрый доступ к основным функциям

### 💹 Таблица курсов
- Реальные данные криптовалют (Bitcoin, Ethereum, Ripple, Litecoin, Cardano, Solana)
- Отображение цены, изменения за 24 часа, рыночной капитализации
- Фильтрация по названию или символу
- Поддержка 3 валют: USD, EUR, RUB
- Кнопка обновления курсов
- Возможность добавления в портфель прямо из таблицы

### 📰 Блог с новостями
- 6 актуальных статей про криптовалюты
- Красивые карточки с эмодзи
- Теги для категоризации
- Интерактивное отображение деталей

### 📧 Контакты
- Полная контактная информация
- Ссылки на социальные сети
- Контактная форма с валидацией
- Уведомления об отправке

### 🔐 Личный кабинет
- **Вход** - авторизация в системе
- **Регистрация** - создание нового аккаунта
- **Мой портфель** - управление активами:
  - Просмотр всех добавленных криптовалют
  - Отображение количества и даты добавления
  - Вычисление общей стоимости портфеля
  - Удаление активов
  - Добавление новых активов через форму

## 🚀 Как использовать

1. Откройте файл `index.html` в браузере
2. Исследуйте таблицу курсов криптовалют
3. Читайте новости в блоге
4. Создайте аккаунт в личном кабинете
5. Добавляйте криптовалюты в свой портфель
6. Отслеживайте стоимость вашего портфеля
7. Свяжитесь с нами через контактную форму
